# Place your logo animation frames here

Please copy all logo AVIF image files into this folder.

The images should be named:
- isologo_clear_00000.avif (idle state)
- isologo_clear_00001.avif
- ... up to ...
- isologo_clear_00012.avif (opening animation end)
- isologo_clear_00013.avif (loop start)
- ... up to ...
- isologo_clear_00095_loop.avif (loop end)

Animation sequences:
- Idle: isologo_clear_00000.avif
- Opening: 00000 → 00012
- Loop: 00013 → 00095_loop
- Closing: 00012 → 00000 (reverse)
